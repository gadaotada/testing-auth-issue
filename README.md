## Getting Started

First, run the development server:
cd ->dir/npm install -> npm run dev //
cd ->dir/backend/ npm install -> npm run start // to run the backend express server on 1337 port :D

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
